# 💻 IT Support Conversational AI (RAG Chatbot)

This project is a conversational AI chatbot built using Retrieval-Augmented Generation (RAG) for IT support queries.

## 🚀 Features
- Context-aware responses
- IT troubleshooting support
- Fast query resolution
- Built with LangChain + Hugging Face

## 🛠️ Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- Hugging Face Models

## 📂 Project Structure
- app.py → Streamlit UI
- utils.py → RAG pipeline
- data.txt → Dataset

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
